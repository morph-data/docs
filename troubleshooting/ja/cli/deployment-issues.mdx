# ビルドエラー

`morph deploy`が失敗した場合は、以下を参考にして問題を特定してください。

#### `Deployment failed: [MESSAGES]`

このエラーはビルドが完了したのちに、Morphのクラウド環境へデータアプリを展開する際に失敗したことを示しています。

なお、ビルドされたイメージが10GBを超える場合はデプロイが失敗します。この場合は、プロジェクトルートの`Dockerfile`を確認して、不要なファイルやディレクトリを削除してください。

分析に利用するソースデータが大きい場合などは、`Dockerfile`には含めずに、MorphのBuiltin Postgresや外部ストレージなどの利用を検討してください。
