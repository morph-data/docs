---
title: 'config'
---

The `morph config` command is used to configure your Morph cloud account in your local environment.

The result will be saved in `~/.morph/credentials`.

```bash Shell
morph config
```

### Arguments

<ParamField body="--profile" type="str">
You can specify the profile name to be registered in `~/.morph/credentials`.
</ParamField>
