---
title: 'config'
---

The `morph config` command is used to configure your Morph cloud account in your local environment.

The result will be saved in `~/.morph/credentials`.

```bash Shell
morph config
```

### Arguments

<ParamField body="--profile" type="str" requered={false}>
Using the `--profile` option allows you to select a specific section in the `~/.morph/credentials` file.
If a section with the same name already exists, a warning will be displayed asking whether to overwrite it.
</ParamField>
