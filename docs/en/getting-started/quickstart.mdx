<Steps>
  <Step title="Install Morph with pip Command">
    ```bash Shell
    pip install morph-data
    ```
  </Step>
  <Step title="Create a Morph Project">
    Use the `morph new` command to create a Morph project.
    ```bash Shell
    morph new morph-starter-app
    ```
  </Step>
  <Step title="Implement Data Processing in Python">
    Implement data formatting, data analysis execution, and chart creation in Python. You can also use SQL if necessary.
  </Step>
  <Step title="Edit the MDX File">
    Add descriptions and components to the MDX file to complete the app.
  </Step>
  <Step title="Start the Morph Development Server">
    Use the `morph serve` command to start the Morph development server.
    ```bash Shell
    morph serve
    ```
  </Step>
  <Step title="Open the App in a Browser">
    Access `localhost:8080` to open the app.
  </Step>
</Steps>
