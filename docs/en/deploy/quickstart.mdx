---
title: 'Deploy to the cloud'
icon: "rocket"
---

Morph allows you to easily deploy data applications and share them with your team.

<Steps>
  <Step title="Login to the cloud and create an API key">
    Create an API key from the Morph home screen.
  </Step>
  <Step title="Set API with CLI">
    Set the API key and workspace with the `morph config` command.
    ```bash Shell
    morph config
    ```
  </Step>
  <Step title="Deploy to the cloud">
    Deploy to the cloud with the `morph sync` command.
    ```bash Shell
    morph sync
    ```
  </Step>
  <Step title="Access the Viewer">
    Access the Viewer to use the deployed data application!
  </Step>
</Steps>
