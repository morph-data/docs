---
title: 'プロジェクトをアップデートする'
icon: "rotate"
---

Morphパッケージの最新版がリリースされた際、既存のプロジェクトにその変更を適応するには、以下のコマンドを実行してください。

```bash Shell
pip install -U morph-data

cd my_project
morph clean -f
morph serve
```

これらのコマンドを順番に実行することで、プロジェクトのアップデートが完了します。

<Note>
  `morph clean -f`コマンドは、プロジェクトのキャッシュを削除し、最新のパッケージに更新します。
  あなたがキャッシュディレクトリに直接変更を加えている場合は、変更が上書きされてしまうので注意してください。
</Note>
