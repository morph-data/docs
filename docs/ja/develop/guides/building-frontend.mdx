---

Morph では、MDXを使用してフロントエンドを構築します。MDXファイルにコンポーネントを配置することで、Web アプリケーションの UI を構築することができます。

## 開発の流れ

Morph でのフロントエンド開発の手順は以下の通りです。

<Steps>
  <Step title=".mdxファイルを作成する">
    `src/pages` ディレクトリに、 `.mdx` ファイルを作成することで、Webアプリケーションのページを作成します。ファイルパスをベースにURLが生成されます。([ファイルベースルーティング](/reference/ja/mdx/file-based-routing)を参照)
    ```bash
    # /yc-companiesにページを作成
    touch src/pages/yc-companies.mdx
    ```
  </Step>
  <Step title="ページのタイトルを追加する">
    `.mdx` ファイルに、ページのタイトルを追加します。ページの最初の見出しがタイトルとして扱われます。
    ```md src/pages/yc-companies.mdx
    # Y Combinatorの企業一覧
    ```
  </Step>
  <Step title="コンポーネントを配置する">
    [コンポーネントリファレンス](https://morph-components.vercel.app/docs/introduction)を参照しながら、あなたのデータアプリに適したコンポーネントを配置してください。([コンポーネント](/reference/ja/mdx/components)を参照)
    ````md src/pages/yc-companies.mdx
    # Y Combinatorの企業一覧

    <DataTable name="yc_companies" height={500} />
    ````
  </Step>
  <Step title="開発サーバーを起動する">
    開発サーバーを起動することで、Webアプリケーションをローカルで確認することができます。
    ```bash
    morph serve
    ```
  </Step>
</Steps>
