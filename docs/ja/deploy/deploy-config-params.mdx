## 環境変数

Morphのフレームワークではコードで使用する環境変数を設定することができます。

環境変数へのアクセスは通常のPythonのコードと同じように行うことができます。

```python
import os
import pandas as pd
import morph
from morph import MorphGlobalContext

@morph.func
def get_env_value(context: MorphGlobalContext):
    # get env variable named "ENV_NAME"
    env_name = os.environ["ENV_NAME"]
    return pd.DataFrame({})
```

### 環境変数の設定方法

<Tip>ローカルのプロジェクトの内に`.env`ファイルがある場合`morph deply`コマンドを実行することで、環境変数が自動的にダッシュボードに同期されデプロイ後の環境で使用することができます。ローカル環境では`.env`ファイルから環境変数が読み込まれます。</Tip>

<Tabs>
  <Tab title="ダッシュボード">
    個別プロジェクトページの「Environment Variables」タブから「Update」ボタンを押下して作成、削除をすることができます。

    登録された値を編集することはできないため更新する場合は一度削除してから再度登録してください。

    <img src="/assets/images/docs/envvars.png" alt="Environment Variables" />

    <Warning>
    環境変数を更新することでデプロイしたプロジェクトにも反映することができますが反映には10秒程度時間がかかるため注意してください。
    </Warning>
  </Tab>
  <Tab title="ローカル">
    プロジェクトのルートに `.env`ファイルを作成して設定をすることでMorphのフレームワークが自動で読み取りを行います。
    ```bash .env
    KEY1=value1
    KEY2=value2
    ```
  </Tab>
</Tabs>

## pipパッケージ

Morphのフレームワークではpipに公開されているPythonパッケージを自由に活用することができます。

## インストール方法 / デプロイ方法

Morphでは構築したプロジェクトで自由にpipパッケージをインストールすることができます。
pipに公開されてあるパッケージを使用する場合はプロジェクト直下にある`requirements.txt`もしくは`pyproject.toml`に使用するパッケージを追加してください。
`morph deploy`コマンド実行時にこれらのファイルが読み込まれデプロイ環境でも追加したパッケージを使用することが可能です。

### pipを使用している場合

```bash shell
pip install {パッケージ名}
```

### poetryを使用している場合

```bash shell
poetry add {パッケージ名}
```
