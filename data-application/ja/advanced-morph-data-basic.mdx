---
title: "バックエンドとの通信"
---

MDX ファイルから Morph バックエンドと通信する方法には、データコンポーネントの使用 と データ API の 2 つがあります。
データ API として、Python や SQL の実行結果を取得するための `loadData()` と、実行するための `postData()` が提供されています。

いずれの場合にも、データ取得のリクエストは、Morph が内部的に利用する Web API によって処理されます。
バックエンドは、リクエストを受け取ると SQL または Python を実行します。

<Note>
  可能であればデータコンポーネントを使うことを強くおすすめします。データアプリで必要とされるコンポーネントが網羅されており、積極的にメンテナンスされています。
</Note>

テーブル表示や、Plotly で描画したグラフの表示など、用意されたコンポーネントが使える場合はそれらを利用してください。
フロントエンドで複雑な処理を行いたい場合には、データ API を用いることを検討してください。
